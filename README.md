# Freelance Dashboard (React + TypeScript)

A mini-dashboard that displays clients, projects, and payments using TypeScript for type safety. This project demonstrates typed models, Context + useReducer, discriminated unions, and reusable typed React components.

## Features
Type-safe data models: Clients, Projects, Payments
 Global state with Context + useReducer and discriminated union actions
Utility functions for filtering, counting, searching, and validation
 Reusable components: `ClientCard`, `ProjectList`, `DashboardStats`
 Sidebar with collapse + mobile behavior (amber/gold gradient style)
 Sample seed data (2 clients, 2 projects, 1 payment)
